## 📌 프로젝트 개요

> **Busted!**  
> *이 프로젝트는 액션 잠입 숨바꼭질 게임으로, Unreal Engine 5를 사용하여 개발되었습니다.*  
> *멀티플레이는 리슨서버를 이용해 구현하였으며, 스팀 API를 이용해 로비, 보이스챗을 구현하였습니다.*  
> *타겟 플랫폼은 PC(Window), 출시 플랫폼은 Steam이며 개발 기간은 약 3개월 입니다.*
